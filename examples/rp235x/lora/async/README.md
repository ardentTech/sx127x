# RP235x Async Examples
These examples were developed and validated with a [Pico 2 W](https://www.raspberrypi.com/products/raspberry-pi-pico-2/) with the [Adafruit RFM95W](https://www.adafruit.com/product/3072) breakout board. For
different hardware combinations, it's important to select the correct frequency and verify the pin mappings.

## Usage
E.g. `$ cargo run --bin tx`
